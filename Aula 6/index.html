Flex Box

São caixas flexiveis que podem se adaptar ao Breakpoints declarados no CSS - é como se fosse uma trabela.

Elementops e atributos trabalhados juntos com o Flex Box

      Display: exibe um elemento
      Flex-direction: direção dos elementos, por padrao declaramos ele em row (linha)
      Flex-wrap: define quebra de linha da caixa
      Flex-flow: atalho para o uso do direction e wrap
      justify-content : conteudo justificado
      aligh-intems: alinhamento de itens dentro do elemento
